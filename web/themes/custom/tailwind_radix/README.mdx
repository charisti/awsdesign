# Tailwind Radix Theme

This Radix 6 subtheme bundles Bootstrap 5 plus Tailwind CSS utilities. Use the npm scripts to compile Sass/JS via Laravel Mix and generate Tailwind output.

## Setup

```bash
nvm use           # optional, if you rely on .nvmrc
npm install
```

Copy `.env.example` to `.env.local` and set `DRUPAL_BASE_URL` for BrowserSync when using `npm run watch`.

## Development Tasks

```bash
npm run dev          # one-time build (Sass/JS + Tailwind)
npm run watch        # watch Sass/JS and Tailwind in parallel
npm run production   # optimized build for deployment
```

Tailwind configuration lives in `tailwind.config.cjs` and the entry file is `src/css/tailwind.css`. Update the `content` array whenever you add new templates so unused utilities purge correctly.
